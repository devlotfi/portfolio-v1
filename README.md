# ğŸ“‚ Project Setup

## Requirements
- Python
- Poetry
- Node JS
- Yarn

## Install

```bash
# Run in server folder
$ poetry install --no-root

# Run in client folder
$ yarn
```

## Development

```bash
# Run in server folder
$ poetry run python manage.py runserver

# Run in client folder
$ yarn run dev
```
