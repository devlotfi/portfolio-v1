# 📂 Project Setup

## Install

Make sure you have "Poetry" package manager installed or run


```bash
$ pip install poetry
```

Then install the dependencies 

```bash
$ poetry install --no-root
```

## Development

```bash
$ poetry run python src/manage.py runserver
```
